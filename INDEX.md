## 💡 Quick Tips

### Customizing Colors
Edit `src/style.css` and change hex codes:
```css
#0a5338 → your-primary-color
#009CF7 → your-accent-color
```

### Building
```powershell
PowerShell -ExecutionPolicy Bypass -File build.ps1
```

### Running
```bash
run.bat
```

### Troubleshooting
1. Check Java version: `java -version` (need 24+)
2. Rebuild if needed: Delete `bin/` and run `build.ps1`
3. Check CSS copied: Verify `style.css` in `bin/` folder

## 🎨 Design Highlights

### Modern Approach
- Apple-like minimalism (clean, simple)
- Material Design (depth, shadows, interaction)
- Glassmorphism (frosted glass effect)
- Professional academic aesthetic

### Visual Elements
- **Hero Banner**: Gradient, 280px, prominent
- **Search**: Centered, modern styling
- **Cards**: Semi-transparent, soft shadows
- **Tables**: Gradient headers, professional rows
- **Buttons**: 5 style variants with interactions

### Typography
- Titles: 28-42px, bold (700)
- Body: 13-14px, regular
- Small: 11-12px, medium weight
- Font: Modern sans-serif stack
